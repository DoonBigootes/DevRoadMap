text basics

break line (br):
basicamente es una nueva linea
este espacio es de un linebreak

enfasis:
para un enfasis, se puede hacer con cursiva (em)
o negrita (strong)

nbsp:

listas

Lista ordenada (ol)

se usa (li) para los items

  1. item 1
  2. item 2
  3. item 3

Lista no ordenada (ul)

se usa (li) para los items

  • item 1
  • item 2
  • item 3

Lista detallada (dl)

se usa (dt) para "titulo" y (dd) para la "descripcion"

item 1:
- desc item 1
item 2:
- desc item 2
item 3:
- desc item 3
